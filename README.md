# TO-DO List App
 
A customizable to-do list app with multiple list styles. I built this project as part of my self-learning journey in web development.
 
## ✨ Features (work in progress)
 
* Create and manage multiple lists
* Three list styles: Basic, Weekly, and Grocery
* Add, edit, and delete tasks
* Add, edit, and delete lists
* Data persistence with localStorage
## 🛠️ Technologies Used
 
* HTML
* CSS
* JavaScript
## 🔮 Future Improvements
 
* Click on a list to load it as active
* Checkbox to mark tasks as completed
* Export and import lists as JSON
* Visual feedback on button interactions
* Responsive design improvements
